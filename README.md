# Cattle Farm Management System

A simple **Cattle Farm Management System** built using **HTML, CSS, and JavaScript**.  
This project allows you to **add, view, edit, delete, and search** animals in your farm and track **total animals and milk production**.

---

## 🐄 Features

- Add new animals with:
  - ID / Name
  - Type (Cow / Buffalo)
  - Age
  - Milk per day
- View all animals in a **list table**
- **Edit** or **Delete** animal records
- **Dashboard** shows:
  - Total Animals
  - Total Milk (Liters)
- **Search Functionality**: Search animals by name
- Data is stored in **localStorage** (persists on browser reload)
